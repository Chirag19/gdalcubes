Detailed Description

This class defines a view how to look at the data including which resolution, which projection, which spatial / temporal window we are intereseted in following analyses.

The class uses "simplified" date calculations, i.e., date differences are based on a given maximum unit / granularity that we are interested in. For instance, the difference between 2018-04-01 and 2016-06-05 can be 2 years, 23 months, or 666 days. Time intervals are currently not implemented.
