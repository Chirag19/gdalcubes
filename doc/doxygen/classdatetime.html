◆ to_string()

std::string datetime::to_string ( datetime_unit  u)
inline

Produce datetime string up to the given unit. The string will possibly include e.g. seconds although the unit is actually lower e.g. days. This function is implemented to interface tools that require full datetime strings and do not work with e.g. "2001-01".
